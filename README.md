# ClassLinker | DandyHacks 2025

ClassLinker is an AI-powered academic planning tool built during the 48-hour DandyHacks 2025 hackathon. It helps students explore course requirements, understand prerequisite relationships, and generate a semester-by-semester academic plan.

## My Role

This was a 4-person hackathon project. My contributions focused on:

- Implementing most of the frontend interface
- Building the course search and course detail interaction
- Developing the CS requirements page and requirement-tree interface
- Building the floating planner panel for selected courses
- Integrating frontend interactions with backend planning outputs
- Participating in backend pipeline discussions around course data parsing, API contracts, and scheduling results
- Preparing and delivering the final demo presentation

## Features

- Interactive course search
- Course detail panel
- CS requirement-tree visualization
- Floating planner panel
- Backend-generated semester planning output
- Prerequisite-aware scheduling logic

## Tech Stack

- Frontend: React, Vite, JavaScript, CSS
- Backend: Python, Flask
- Data / Logic: JSON course data, prerequisite graph modeling
- Tools: Git, GitHub

## Running Locally

### Backend

```bash
pip install -r requirements.txt
python app.py
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
